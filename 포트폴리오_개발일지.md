# 포트폴리오 개발일지

**2026.07.10**

오늘 하루 만에 포트폴리오 웹페이지(`index.html`)를 처음부터 만들고, 계속 다듬었다. 작업한 순서대로 기록해둔다.

1. **자기소개 문구 작성**
   포트폴리오 첫 화면에 넣을 담백한 자기소개 3줄을 먼저 썼다. 아직 진로를 고민 중이라는 것, AI·XR로 관심사를 기록하고 있다는 것을 과장 없이 담았다.

2. **작품 설명 초안 작성**
   작품마다 제목 + 2줄 설명을 관람객에게 말 걸듯 부드러운 톤으로 작성. 어떤 도구로 어떻게 만들었는지, 어떤 마음으로 만들었는지가 같이 드러나게 썼다.

3. **포트폴리오 웹페이지 첫 버전 제작**
   모던한 다크 테마로 첫 화면(프로필 + 자기소개), 작품 섹션, 테마곡 섹션, VR 링크 버튼을 구성. 폴더에 미리 넣어둔 실제 이미지·영상·음원·3D 파일(glb)을 전부 찾아서 직접 연결했다 — 자리표시자가 아니라 진짜 파일로.

4. **AI 프로필 사진 작품 재정리**
   "다섯 개의 표정, 그리고 하나" → **"하나의 표정, 다섯 가지 그림체"**로 제목·설명 변경. 다른 표정이 아니라 같은 표정을 다섯 가지 그림체로 그려본 것이라는 점을 명확히 했다.

5. **맛집 지도 프로젝트 콘텐츠 반영**
   AI 이미지 3장을 맛집 지도 웹사이트 홍보용 포스터·카드뉴스로 재소개. 진행 중인 맛집 지도 웹사이트([matzip-map-nine.vercel.app](https://matzip-map-nine.vercel.app/))를 여섯 번째 작품으로 추가하고 바로가기 버튼 연결. 이후 웹사이트 스크린샷도 받아서 카드에 반영.

6. **필름 다이어리 설명 수정**
   실제 하루 기록이 아니라 AI로 만든 가상의 먹방 다이어리 영상이라는 점을 반영해 제목을 **"존재하지 않는 어느 날의 먹방"**으로 바꿨다.

7. **3D 모델 뷰어 이슈 발견 및 해결**
   `model-viewer`가 로컬(`file://`)로 파일을 열면 브라우저 보안 정책 때문에 glb를 못 불러와 빈 화면으로 보이는 문제를 발견. 로컬에서 열면 자동으로 스크린샷으로 대체되고, 온라인(http/https)으로 배포하면 자동으로 인터랙티브 3D 뷰어(드래그 회전)로 전환되도록 분기 처리했다. 이번 발표는 로컬로 진행 예정이라 당장은 스크린샷으로 보임.

8. **제목·테마 리뉴얼**
   포트폴리오 제목을 **"일단 만들어봤어요 — 신유진의 AI×XR 실험 노트"**로 변경. "고민/질문" 콘셉트보다 "직접 만들어본 실험" 쪽에 더 가까워졌다고 판단해서. 색상 테마도 보라·청록에서 **오렌지·코랄 계열 웜톤**으로 교체 — 맛집 콘텐츠와도 잘 어울림.

9. **테마곡 섹션 개선**
   곡마다 그라데이션 앨범 아트 + 재생 버튼을 얹어서, 텍스트만 있던 밋밋한 카드에 이미지 요소를 추가했다.

10. **AI 프로필 사진 인터랙션 추가**
    그림체 선택 버튼(수채화 / 파스텔 일러스트 / 3D 캐릭터 / 픽셀아트 / 유화)을 만들어서, 마우스를 올리면 해당 그림체로 미리보기, 클릭하면 그 그림체로 고정되도록 구현. (다시 클릭하면 통합본으로 복귀)

11. **이미지 레이아웃 통일**
    처음엔 이미지가 잘리지 않게 원본 비율 그대로 보이도록 수정했다가, 카드마다 크기가 들쑥날쑥해 보여서 다시 **1:1 정사각형 프레임 + 여백(object-fit: contain)** 방식으로 통일. 잘리지도 않고, 크기도 맞춰짐.
