# 신유진 · AI×XR 포트폴리오

AI와 XR로 만든 여섯 가지 작품과 테마곡을 소개하는 개인 포트폴리오 페이지입니다.
빌드 도구 없이 `index.html` 한 파일로 이루어진 정적 웹페이지예요.

## 구성

- `index.html` — 전체 페이지 (Intro / Works / Theme)
- `1. AI 프로필 사진/` ~ `7. 웹사이트/` — 각 작품에 쓰인 이미지·영상·음원·3D 파일
- `포트폴리오_개발일지.md` — 제작 과정 기록

## 로컬에서 보기

`index.html`을 브라우저로 그냥 열면 됩니다. 3D 오브젝트(`model-viewer`)는 브라우저 보안 정책 때문에 로컬(`file://`)에서는 스크린샷으로 대체되어 보이고, 온라인으로 배포하면 자동으로 드래그 회전이 되는 3D 뷰어로 전환됩니다.

## 배포

GitHub + Vercel로 배포합니다. 정적 페이지라 별도 빌드 설정 없이 리포지토리를 그대로 Vercel에 연결하면 됩니다.
